# posicaoCelular

- Instalar o Expo CLI (se você não tiver o Expo instalado):
npm install -g expo-cli

-Instalar dependências do acelerômetro:
A biblioteca expo-sensors é usada para acessar os sensores, como o acelerômetro.
npm install expo-sensors

-Instale as dependências:
npm install

-Rode o projeto:
npx expo start
